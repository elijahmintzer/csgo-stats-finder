What is this?

This was a simple project developed for the purpose of learning how full-stack apps work.

With this application, you are able to input a user's SteamID and get their CS:GO stats.

What can you use this for?

Like I said, you can get user's stats with this simple app.

The code is completely open-source at my github repository.

Feel free to take a peek in and look at the code.

How did I make this?

I made this using Javascript, and NodeJS.

I used ExpressJS for the server and clientside Javascript for inputting the stats to the user.

I then used Steam's API to fetch the stats from Steam.

What else do I intend to do with this?

I don't plan to do anything else with this project, the goal has been fulfilled.

This project was only for learning purposes and nothing else.
